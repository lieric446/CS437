# FinalProject

## Overview

This project implements a cloud-based system where the server handles processing tasks and facilitates communication between a mobile app and a Raspberry Pi device.

## Components

- **app/**: Mobile application built with Expo (React Native)
- **rpi/**: Raspberry Pi scripts and utilities (Python)
- **server/**: Cloud server for processing and communication facilitation

## Architecture

The server acts as the central hub:
- Provides cloud processing capabilities
- Manages communication between the mobile app and Raspberry Pi
- Handles data flow and coordination between components
